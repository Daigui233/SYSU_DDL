# Master_RK3588S 上位机说明

## 一、目录用途

本目录用于存放 `SYSU_DDL` 项目的上位机工程。

当前阶段，上位机工程尚未正式开始搭建，因此本目录暂时作为占位目录保留，用于保证仓库目录结构完整，并为后续协同开发预留统一入口。

## 二、当前状态

当前上位机部分暂未接入正式代码，后续将围绕以下内容逐步开展：

- 视觉感知与图像处理
- OCR 路牌识别
- 红绿灯与限速标志识别
- 行人、车辆、障碍物等元素识别
- 路线规划与行为决策
- 与 `TC264D` 下位机的串口通信

## 三、后续开发约定

后续上位机工程正式接入后，建议遵循以下原则：

- 与下位机保持明确接口边界
- 通信协议统一管理
- 感知、决策、通信模块尽量解耦
- 每个功能独立提交并审查

## 四、补充说明

当前下位机为优先开发对象，详见仓库根目录 `README.md` 以及 `Slave_TC264D/README.md`。

本文件仅用于保留上位机目录并说明当前状态，后续会随着上位机工程推进持续更新。
